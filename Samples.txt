2. Update Record (Upsert)

URI path: /dictionary
HTTP method: PUT

Sample:
{
    "derivatives": [
      "aurae","auras"
    ],
    "meaning": [
      "1.the distinctive atmosphere or quality that seems to surround and be generated by  persons, thing, or place."
    ],
    "relatedWords": [
      "atmosphere","ambience"
    ],
    "type": "noun",
    "word": "aura"
}
---------------------------------------------------------------------------------------------------------------------------------------------------

3. Get all Records
   URI path: /dictionary
   HTTP method: GET
   
----------------------------------------------------------------------------------------------------------------------------------------------------

4. Get records by letter
   URI Path:/dictionary/letter/{letter}
   HTTP method: GET
   Sample: /dictionary/letter/A
   
-----------------------------------------------------------------------------------------------------------------------------------------------------

5. Get records by word and optional type
   URI path:/dictionary/word/{word}?type={type}
            /dictionary/word/{word}
   HTTP method: GET
   Sample:/dictionary/word/aura?type=noun
          /dictionary/word/aura
---------------------------------------------------------------------------------------------------------------------------------------------------

6. Delete word and optional type
   URI path:/dictionary/word/{word}?type={type}
            /dictionary/word/{word}
   HTTP method: DELETE
   Sample:/dictionary/word/aura?type=noun
         /dictionary/word/aura
		 
-------------------------------------------------------------------------------------------------------------------------------------------------
7. Delete words for a letter
   URI Path:/dictionary/letter/{letter}
   HTTP method: Delete
   Sample: /dictionary/letter/A
